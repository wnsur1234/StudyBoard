# 📋 게시판 프로젝트

> 혼자 공부해보기 위해 만드는 CRUD 게시판 프로젝트입니다.  
> Spring Boot + Thymeleaf를 활용하여 화면까지 구성하였습니다.

---

## 🛠 기술 스택
- **Language** : Java 17
- **Framework** : Spring Boot 3.x
- **Template Engine** : Thymeleaf
- **Database** : MySQL 8.x
- **ORM** : Spring Data JPA
- **Build Tool** : Gradle
- **Etc** : Lombok, Spring Validation

---

## 📌 주요 기능
- 게시글 등록, 조회, 수정, 삭제 (CRUD)
- 타임리프 기반의 간단한 UI
- MySQL 연동
- (추후) 로그인 및 회원 기능 추가 예정
